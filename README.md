# The-Abstraction-Server
- Công nghệ sử dụng
  - ASP.NET Core 8.0
  - C#
  - Clean Architecture
  - CQRS Pattern
  - Identity (Role and User Management)
  - SQL Server, MonggoDB
  - MongoDB.Driver, MongoDB.BSON
  - Dapper
  - Entity Framework
  - AutoMapper
  - MediatR
  - JWT Authentication and Authorization
- Dự án cung cấp các API cho các ứng dụng như web, app, Unity...
